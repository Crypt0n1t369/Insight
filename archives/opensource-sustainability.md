# Open Source Sustainability Models

## 1. GitHub Sponsors
- Direct funding to maintainers
- Platform takes 0% (as of recently)
- Works for individuals

## 2. Open Collective
- Transparent budgeting
- Collective ownership
- Anyone can expense

## 3. Patreon/ko-fi
- Monthly support
- Creator benefits
- Platform takes cut

## 4. Bounties
- Issue bounties (Gitcoin)
- Pay for specific work
- Market for skills

## 5. Foundation
- Apache, Linux, Mozilla
- Corporate + individual donors
- Long-term sustainability

---

## For Credo: Hybrid

- **Phase 1:** Grants (foundations)
- **Phase 2:** Open Collective style
- **Phase 3:** GitHub Sponsors for top contributors
- **Never:** Advertising, selling data
