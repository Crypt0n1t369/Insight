**How it works:** Candle sales. The project has been running for years (5+ years based on the U.S. Embassy mentioning their anniversary). It has generated significant public visibility — including an exhibition at Riga Airport.

**The "losing traction" signal:** From search results, the project appears to have had high-profile moments (Riga Airport exhibition, U.S. Embassy visit, Embassy charity auction) but these are event-based rather than sustained ongoing engagement. **What this means for the platform:** This is a real, ongoing project with a concrete need (fundraising for a physical home) and a documented track record. A young person contributing to this project would be contributing to something tangible — a building with 16 people living in it.

**Website/Facebook:** facebook.com/millioncandlefor (verified, content not accessible via web fetch)

**Gap this fills for us:** Creative track challenge (Real Challenge type — the candle campaign needs creative reinvigoration). Impact track (contributing to a real social cause). Portfolio piece (the campaign is public, named).
