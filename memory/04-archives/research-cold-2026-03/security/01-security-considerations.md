# Security & Privacy Research

## Security Considerations

### 1. Data Privacy
- Audio sessions are sensitive
- Minimize storage
- Encryption at rest

### 2. Anonymous Identity
- Prevent Sybil attacks
- No linking to real identity
- ZK proofs for verification

### 3. AI Safety
- Prompt injection prevention
- Output filtering
- Rate limiting

### 4. Financial
- No stored payment data
- Use Stripe/standard processors
- Tax compliance

## Privacy by Design

### Data Minimization
- Only collect what's needed
- Delete unused data
- Anonymize for research

### User Control
- Export all data
- Delete account + data
- Opt-out of analytics

### Transparency
- Open source (where possible)
- Clear policies
- Regular audits
