## Adding New Domains

1. Create `memory/domains/[domain-name].md`
2. Add entry here with path, description, status
3. Populate with initial notes, questions, known resources
4. Update as learning accumulates

## Domain Recall Protocol

When working on a task:
1. Identify relevant domain(s) via memory_search
2. Load domain context into working memory
3. Update domain file with new learnings
4. Cross-link related domains
