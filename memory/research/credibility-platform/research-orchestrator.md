## KEY FINDINGS SUMMARY

### Technical Architecture
- Frontend: React + Next.js
- Backend: Node.js + GraphQL
- Database: PostgreSQL + Redis
- Storage: IPFS + Pinata
- Identity: Wallet-based (Ceramic DID)
- Blockchain: Polygon/Solana for credibility

### Credibility System
- Weighted scoring: contribution × impact × scarcity
- Tiers: Newcomer → Contributor → Trusted → Elder
- Anti-gaming: ZK proofs for uniqueness
- Decay: Optional temporal decay

### Governance
- Quadratic voting
- Optional: Liquid democracy (delegation)
