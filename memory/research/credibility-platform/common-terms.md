# Credo - Common Terms & Definitions

## Core Concepts

| Term | Definition |
|------|------------|
| **Branch** | A version of a research/project that can be forked and merged |
| **Credibility** | Reputation score earned through value-based contributions |
| **Fork** | Creating a new branch from an existing one |
| **Merge** | Combining branch changes back into parent |
| **Contribution** | Any action that earns credibility (edit, comment, review, etc.) |
| **Credibility Threshold** | Minimum credibility needed to perform certain actions |
| **Slashing** | Penalty for violating community rules |
| **Quadratic Voting** | Voting weight = √(votes) to prevent whale capture |
| **ZK Proof** | Zero-knowledge proof for identity verification |
| **Sybil Attack** | Fake users flooding the system |

| Term | Agent | Definition |
|------|-------|------------|
| **Flow State** | Gamification | Optimal engagement state (Csikszentmihalyi) |
| **SDT** | Gamification | Self-Determination Theory (autonomy, competence, relatedness) |
| **Semaphore** | Security | ZK protocol for anonymous identity |
| **Sortition** | Alternatives | Random selection for governance |
| **Dunbar Number** | Alternatives | ~150 - max stable social relationships |
