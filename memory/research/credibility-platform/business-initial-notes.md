## Current Thinking

### Recommended Model: Open Core + Grants + DAO

**Free Tier (Individuals):**
- All core functionality
- Credibility system (can't be bought)
- Basic avatar
- Public branches

**Paid Tier (Organizations):**
- Private branches
- Analytics
- API access
- White-label
- Support

**DAO/Treasury:**
- Token for governance (NOT credibility)
- Revenue goes to treasury
- Grants distributed to community

**Grants:**
- Foundation funding for social good
- Covers development costs
- No pressure to monetize users
