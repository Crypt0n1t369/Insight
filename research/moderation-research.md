# Content Moderation in Anonymous Systems

## The Challenge
Anonymous = hard to moderate. But Credo needs SOME moderation for safety.

## Approaches

### 1. Decentralized Moderation (Recommended)
- User flags content
- Random jury of TRUSTED+ reviews
- Slashing for false flags
- Appeals process

### 2. Algorithmic
- AI detects spam/abuse
- Auto-slow for new accounts
- Rate limiting
- Pattern detection

### 3. Community Norms
- Clear code of conduct
- Upfront acceptance
- Self-moderation
- Fork away from bad actors

## For Credo: Hybrid Approach

1. **AI first pass** - Detect obvious spam
2. **User flags** - Community reporting
3. **Jury review** - Random TRUSTED+ for contested
4. **Slash bad actors** - Remove credibility
5. **Fork escape** - Leave bad communities

---

## Stack Exchange Model
- Flag → Close Votes → Delete Votes
- Reputation-gated actions
- Court of meta (appeals)
- Works WITHOUT anonymity

## For Credo (with anonymity)
- Need MORE trust before moderation power
- Slashers must be stake-holders
- Appeal process essential
